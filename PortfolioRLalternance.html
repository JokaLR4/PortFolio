Préparation Matériel et déroulé :

Monter l'afficheur client suivant la cahier des charges
Mise à jour Windows update et driver
Renommer la Caisse enregistreuse en " Caisse-01 ou nom du client"
Aller dans le NAS de l'entreprise dans un dossier nommé "prépa-caisse récupérer les logiciels"
Installation des packages logiciels Google, Teamviewer, SumatraPDF etc..
Crée un dossier dans le lecteur C pour regrouper les logiciels
Crée un sous-dossier "Télémaintenance + l'ID" de Teamviewer et le faire pointer dans la barre d'outils
Changer le Mot de passe pour en mettre un sécurisé avec minimum une majuscule, un caractère spécial, un chiffre.
Règle de pare-feu : Récupérer sur le NAS, le fichier parefeu.wfw faire l'import dans le pare-feu avancé
Installer le Logiciel selon le cahier des charges : LEO / INNOSHOP / CRISALID
Installation de l'imprimante en Port COM selon la possibilité ou RJ45(ethernet 8 fils)
Installation du tiroir caisse relié a l'imprimante en RJ-11 (4fils)
L'Antenne Wifi se paramètre ensuite (Unifi) changer le DNS wifi et le mot de passe sécurisé
Le PAD se configure en ajoutant l'adresse IP dans le Logiciel de léo (Léopocket) une synchronisation entre la caisse et le PAD s'éffectue
